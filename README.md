# service-tandhur-v1

REST API for service-tandhur.

# Requiments:

- Node.JS

# Installation:

Dev Enviroments:

```cmd
$ cd service-tandhur-v1
$ npm install
$ npm run dev
```

Production Enviroments:

```cmd
$ cd service-tandhur-v1
$ npm install
$ npm start
```

Run on

```cmd
localhost:3003
```
